## Авторизация
### POST sign up
`POST /api/auth/signup`

input:
```json
{
	"username":"username",
	"firstname":"firstname",
	"lastname":"lastname",
	"email":"useremail@mail.ru",
	"phone":"89528125252",
	"password":"password12345"
}
```
output:
```json
{
  "message": "User registered successfully!"
}
```

### POST sign in
`POST /api/auth/signin`

input:
```json
{
	"username":"username",
	"password":"password12345"
}
```
output:
